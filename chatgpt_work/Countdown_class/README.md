以下の要件に基づいて Countdown クラスを作成してください。

## プロパティ:

initialValue(初期値、数値)

## コンストラクタ:

コンストラクタは initialValue を引数として受け取り、そのプロパティを初期化します。

## メソッド:

**decrement():**  
initialValue を 1 ずつ減らします。

**reset():**  
initialValue を元の値にリセットします。

**getValue():**  
現在の initialValue を返します。
